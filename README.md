# Django Movie Platform

A Django-based movie platform where users can browse movies, view details, search,add favourites ,post comments and manage their profile.

## Features
- User Authentication (Register/Login/Logout)
- Browse movies & view movie details
- Search movies
- Categories
- User profile
- Admin panel for managing content

## Tech Stack
- Python
- Django
- HTML/CSS
- SQLite (default) 
- Git & GitHub
